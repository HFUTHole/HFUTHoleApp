# HFUTHole 工大树洞前端

一个合工大的匿名论坛平台

## 为什么不用pnpm

经历过长时间的挣扎后，pnpm和npm安装依赖丢给expo服务器打包的时候是最容易报错的，不是peerDependencies没安装正确的版本就是安装的过程直接报错,
所以最终选择了yarn

虽然我讨厌yarn，但是我更讨厌报错
